# Cyber Glitch Theme

A bold, avant-garde theme for programmers.

## Features
- **High Contrast Colors**: Neon Green, Hot Pink, Electric Blue on Deep Black.
- **Rich Animations**: Glitch effects, typing text, hover transformations.
- **Matrix Rain**: A subtle code rain effect in the background.
- **Responsive Design**: Looks great on all devices.

## Configuration
You can customize the colors and glitch intensity in the theme settings.
